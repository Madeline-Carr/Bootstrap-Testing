Grids

The grids in Bootstrap are very easy to use. Bootstrap divides the page into 12 columns that the grid is based on. To tell an element what size it should be you use the col- and then pick a size of the screen and how large the element should be. The sizes are xs, s, md, lg, and xl. The next peice of information is a number between 1 and 12.

So for example, an element with the class of "col-md-6" would take up half the width of the screen on medium sized screens like tablets.

I am a div with the class of "col-md-12".
I am a div with the class of "col-md-6".
I am a div with the class of "col-md-3".

Also you should know that if you make a class for a lower screen size it will apply to greater screen sizes.

So I am a div with the "col-md-6" but I will still only take up 6 columns on larger screens.

Buttons

There are many diffrent button options. To make a Bootstrap button you need to add the class of btn to a button. There are also colors you can use with button classes. For example if you have a button with the class of "btn btn-primary" then that button will have a blue background and white text. Replacing primary with dark will make a dark button, light will make a white button, and danger will make a red button.

The same classes can be used on a and input tags.

You can also assign classes like "btn btn-outline-primary" which will give you an outline in the color.

To change button size you can use "btn-lg" and "btn-sm" to make buttons larger or smaller than their original size.

To make a button take up the entire width of the parent you can use the "btn-block" class.

Button Groups

Using the "btn-group" class on a div with buttons in it will create a block of buttons together!

If you want them to stack on top of each other use the "btn-group-vertical" class.

You can change the size of the button group by using the "btn-group-lg" and "btn-group-sm" classes.

Closing Alerts

To close an alert you can add an x button. To do that make an alert with the class of "alert" and "alert-dismissible." Then inside of that div make a button with the class of "close" and the data-dismiss set to "alert." You can also add the "fade" or "show" classes to edit how it appears or disappears.

In the button you should write "×." This is better than just using an x.

Tables

There are many diffrent options for Bootstrap tables, adding the class of "table" to the table tag will add the Bootstrap styling and adding the class of "table-striped" will make every other row darker. Normally the tables are light but you can make them dark with the "table-dark" class. You can also add the "table-bordered" class to have a boarder around the table or you can use a "table-borderless" class to remove all borders. The "table-hover" class will make rows darker when you hover over them or lighter if it is already dark. You can also use the "table-sm" class to make the table smaller.

Dropdowns

To make a dropdown you make a div with the "dropdown" class then you make a button that can activate the dropdown. The button should have a type of button and a class of "dropdown-toggle" and the attribute data-toggle set to "dropdown." After the button there should be another div with the class of "dropdown-menu" and inside of that there should be the content for the dropdown. Each item in the dropdown should have the class of "dropdown-item."

The class of "dropdown-header" makes header in the dropdown and the class of "dropdown-divider" makes dividers.
