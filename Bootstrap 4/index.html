Buttons

There are many diffrent button options. To make a Bootstrap button you need to add the class of btn to a button. There are also colors you can use with button classes. For example if you have a button with the class of "btn btn-primary" then that button will have a blue background and white text. Replacing primary with dark will make a dark button, light will make a white button, and danger will make a red button.

The same classes can be used on a and input tags.

You can also assign classes like "btn btn-outline-primary" which will give you an outline in the color.

To change button size you can use "btn-lg" and "btn-sm" to make buttons larger or smaller than their original size.

To make a button take up the entire width of the parent you can use the "btn-block" class.

Button Groups

Using the "btn-group" class on a div with buttons in it will create a block of buttons together!

If you want them to stack on top of each other use the "btn-group-vertical" class.

You can change the size of the button group by using the "btn-group-lg" and "btn-group-sm" classes.

Collapsing

In the example below there is a hidden div. It has the class of "collapse" so it does not initially appear on the page. It could also have a class of "collapse show" which would make it appear at the start. It also has an id of "demo" so it can be used with a button.

The button has an attribute called data-toggle which is set to "collapse" and a data-target attribute that is set to "#demo". The data-target class is needed so the button knows what to show.
